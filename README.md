# infotech-test-task

## Стек
- React 19 + TypeScript
- Vite 7
- ESLint
- `lucide-react` (иконки)

## Методология
- Feature-Sliced Design: `shared` / `entities` / `widgets`.
- Компонентный подход и типизация через TypeScript.

## Реализовано
- Таблица пользователей.
- Поиск и фильтры.
- Пагинация.
- Ресайз колонок.
- Виртуализация строк.
- Модальное окно с деталями выбранного пользователя.
- Базовая обработка HTTP-ошибок.

## Полезные команды
```bash
yarn install      # установка зависимостей
yarn dev          # запуск в режиме разработки
yarn build        # production-сборка
yarn lint         # проверка линтером
yarn preview      # локальный просмотр production-сборки
```
